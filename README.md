# 📊 AI-Powered Job Market Analytics Dashboard

## 🚀 Project Overview

This project is an interactive Job Market Analytics Dashboard built using Microsoft Excel. The dashboard analyzes 20,000+ job postings and provides insights into hiring trends, salary distribution, remote work opportunities, and experience requirements across different countries and job roles.

The project demonstrates practical data analysis, dashboard development, and business intelligence skills using Excel.

---

## 🎯 Objectives

- Analyze global job market trends.
- Identify hiring patterns across countries.
- Understand salary distribution for different roles.
- Measure remote work opportunities.
- Track experience requirements for job positions.
- Build an interactive dashboard for data-driven insights.
